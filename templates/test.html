Click the button below to test Apple SSO authentication:

Sign in with Apple

Available Endpoints:

GET /social-auth/login/apple-id/ - Start Apple login
GET /social-auth/complete/apple-id/ - Apple callback
GET /auth/user/ - Get current user info
POST /auth/logout/ - Logout user

Configuration Required:

  1. Update SOCIAL_AUTH_APPLE_ID_CLIENT in settings.py
  2. Update SOCIAL_AUTH_APPLE_ID_TEAM in settings.py
  3. Update SOCIAL_AUTH_APPLE_ID_KEY in settings.py
  4. Add private key to SOCIAL_AUTH_APPLE_ID_SECRET
  5. Configure redirect URL in Apple Developer Console
